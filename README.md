# My-excel-Projects
📊 Excel Retail & Sales Analysis — Data Technician Bootcamp Projects: This repository contains a collection of Excel projects completed during my Data Technician Bootcamp, showcasing practical skills in data cleaning, analysis, reporting, and visualisation using Microsoft Excel.
🚀 Project Overview: Across multiple datasets and lab exercises, I worked with retail, sales, and bike‑store data to:
• 	Clean and organise raw data
• 	Apply essential Excel formulas
• 	Build PivotTables and PivotCharts
• 	Categorise data using logical functions
• 	Explore insights through filtering, sorting, and visualisation

🧰 Skills Demonstrated: Excel Formulas
Used throughout the projects to calculate totals, averages, and lookup values:
- SUM(), SUMIF()
- AVERAGE(), AVERAGEIF()
- DATE(), MONTH(), YEAR()
- UNIQUE()
- VLOOKUP()
- 🔍 Data Cleaning & Exploration
- Sorting values (e.g., Age largest → smallest)
- Filtering datasets to isolate key segments
- Identifying duplicates and unique values
- Using conditional logic to classify data

📈 PivotTables & PivotCharts
Created PivotTables to summarise:
- Sales by county and product
- Customer demographics
- Market performance across countries
- Profitability by age group and gender
Charts were used to visualise trends and support insights.

🔄 Logical Categorisation with SWITCH()
Used the SWITCH() function to classify sales volumes:
=SWITCH(TRUE, C2 > 600, "High", C2 >= 300, "Medium", "Low")
